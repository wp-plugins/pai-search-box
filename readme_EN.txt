== Description ==

The PAi Search Box widget provides a quick way for users to perform searches in the Páginas Amarelas site (http://www.pai.pt) directly from your blog.

You can customize the size of the search to match it to your theme. Edit the main file 'pai_searchbox.php' to see more details
.
Please note that by using this widget you agree maintain the main aspect of the search box - logo, background images and colors - in order to respect the branding of Páginas Amarelas, SA.

For more information please visit http://www.pai.pt/content/staticpages/PAi_novo/pt_pai_noseusite.html

== Installation ==

1. Extract the archive in your computer (you may use the "Extract here" option)
2. Upload "pai_searchbox.php" to the "/wp-content/plugins/" directory
3. Activate the plugin through the "Plugins" menu of you WordPress administration panel
4. Go to the "Widgets" menu under "Themes" the drag the "PAI Search Box" widget to the sidebar
