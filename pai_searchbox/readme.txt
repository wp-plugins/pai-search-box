Este widget coloca uma caixa de pesquisa que permite aos utilizadores pesquisar contactos de Empresas, Servicos e Particulares directamente nos sites http://www.pai.pt e http://www.pbi.pt a partir de um blog Wordpress.

== Description ==

O Widget PAi Search Box oferece aos utilizadores uma forma rapida de efectuar pesquisas no site Paginas Amarelas e Paginas Brancas (http://www.pai.pt e http://www.pbi.pt) directamente a partir do seu blog.

Pode personalizar o tamanho da caixa de pesquisa para que se adapte melhor ao tema que utiliza no seu blog depois de instalar.

Note que ao utilizar este widget concorda em manter o aspecto original da caixa de pesquisa - logotipo, imagens de fundo e cores - de modo a respeitar a imagem da marca Paginas Amarelas, SA.

Para mais informacoes visite http://www.pai.pt/content/staticpages/PAi_novo/pt_pai_noseusite.html

== Installation ==

1. Extrair o arquivo ZIP para o seu computador (pode utilizar a opcao "Extrair aqui")
2. Faca upload de toda a pasta "pai_searchbox" para a pasta "/wp-content/plugins/" do seu blog. Caso utilize o Wordpress 2.7 podera fazer upload do ficheiro ZIP directamente do painel de administracao
3. Active a extensao na seccao "Extensoes" do painel de administracao do seu blog
4. Na seccao "Widgets" do menu "Apresentacao" arraste o widget "PAI Search Box" para uma das barra laterais e va a "Editar" para configurar
